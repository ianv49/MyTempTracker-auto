# 📱 iPhone Temp Tracker
A modern, techie dashboard to monitor device temperatures using GitHub Pages.

## 🚀 Features
* **Neon Dashboard:** Dark mode UI with neon blue gradients.
* **Goal Line:** Visual indicator (dotted red) for 30°C threshold.
* **Max Highlight:** The highest recorded temperature automatically turns red.
* **Scrollable:** View history by swiping the chart.

## 📖 User Guide
### How to Log Data
1. Open your GitHub repository.
2. Edit `data.txt`.
3. Add a new entry using the format: `|Time,Temp` (Example: `|20:30,28`).
4. Commit the changes.

### How to View
* Open your [GitHub Pages Link].
* Use the **Refresh Chart** button to pull the latest logs from Git.
* Tap **View Log** to see the raw text file.
